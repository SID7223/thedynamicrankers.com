# Operations Hub & Internal Command Center

## Industry-Level Infrastructure
This project features a professional-grade internal dashboard for task management, threaded communication, and CRM operations. It is built on Cloudflare Pages and D1 (SQLite).

### Core Components
- **Tasks**: Strategic directives with human-readable IDs (e.g., TASK-101) and full-text search.
- **Messaging**: Threaded "Strategic Comms" with task-based and global rooms.
- **CRM**: Client profiles, automated invoices, and appointment scheduling.
- **Security**: Operative-level authentication for SID and Eric.
